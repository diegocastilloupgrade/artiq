# Drafts Module

This module handles article draft generation and editing.

## Features

- Draft creation from topic + product
- AI-assisted content generation
- Draft editing and saving
- Partial regeneration of sections
- Draft-to-CMS publication

## Entities

- `ArticleDraft` - Draft article record
- `ProductPiece` - Product-specific content piece

## Services

- `DraftService` - Draft management
- `GenerationService` - AI content generation

## Controllers

- `DraftController` - Draft CRUD and generation endpoints
