# Analytics Module

This module handles GA4 data retrieval and analytics summaries.

## Features

- GA4 data retrieval
- Article performance metrics
- Affiliate click tracking
- CTR calculation
- Analytics snapshots

## Entities

- `AnalyticsSnapshot` - Analytics data for an article

## Services

- `AnalyticsService` - GA4 integration and data retrieval

## Controllers

- `AnalyticsController` - Analytics dashboard endpoints
