# Topics Module

This module handles topic management, import from external sources, and core CRUD operations.

## Features

- Manual topic creation
- Import from trend sources (Google Trends, etc.)
- Topic listing and filtering
- Topic status management
- Priority marking

## Entities

- `Topic` - Core topic entity

## Services

- `TopicService` - Business logic for topic management

## Controllers

- `TopicController` - HTTP endpoints
