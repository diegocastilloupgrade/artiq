Summary: Authentication & Roles (T-1.03)
Description:
OpenSpec Reference: T-1.03
Source: openspec/changes/build-mvp-v1/tasks.md

Objetivo:
Implementar autenticación para backoffice con control por roles.

Acceptance Criteria:
- Implementar autenticación JWT para usuarios de backoffice.
- Definir y aplicar roles: admin, redactor, lector.
- Restringir acceso al backoffice solo a usuarios autenticados.
- Implementar validaciones de rol para funcionalidades solo-admin.

Implementation Notes:
- Módulo principal: backend/src/auth.
- Incluir guards y decorators reutilizables para roles.
- Añadir endpoint de login que retorne token JWT.

Labels: openspec,mvp-v1,backend,auth

Summary: Define Domain Entities (T-2.01)
Description:
OpenSpec Reference: T-2.01
Source: openspec/changes/build-mvp-v1/tasks.md

Objetivo:
Definir el modelo de dominio base del MVP y sus invariantes.

Acceptance Criteria:

Labels: openspec,mvp-v1,backend,domain-model

Summary: Database Setup & Migrations (T-2.02)
Description:
OpenSpec Reference: T-2.02
Source: openspec/changes/build-mvp-v1/tasks.md

Objetivo:
Configurar persistencia y esquema inicial de base de datos.

Acceptance Criteria:
